# TÖL103M – Assignment P1

**Course:** TÖL103M – Fall 2024  
**Instructors:** Hyytiä and Ágústsson  

## Description
This project implements a simple serial interface service for an ESP-based device.

## Commands
- `MAC` – prints device MAC address  
- `ID` – prints student email addresses  
- `DEC <number>` – converts binary, octal, hex, or decimal to decimal  
- `STATUS` – prints system uptime, available cores, and free heap memory  

## Notes
- Commands and responses are case-insensitive  
- Prompt format: `> `
- Newlines use `\n`
